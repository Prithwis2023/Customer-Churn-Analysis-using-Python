# Customer-Churn-Analysis-using-Python

### Dataset

<a href="https://www.kaggle.com/datasets/blastchar/telco-customer-churn">Telcho Churn Analysis<a>


##### Questions -
Load the dataset and display the first 5 rows.

Show the number of rows and columns in the dataset.

Display the data types of all columns.

Check if there are any missing values in the dataset.

Show the number of unique values in each categorical column.

Find the percentage of customers who have churned vs. not churned.

Plot a bar chart showing churn count for each gender.

Show the distribution of customer tenure using a histogram.

Compare the average MonthlyCharges of churned vs. non-churned customers.

Find which InternetService type has the highest churn rate.

Plot a boxplot comparing MonthlyCharges and Churn.

Show the correlation between tenure, MonthlyCharges, and TotalCharges.

Does Contract type affect churn rate? Show with a plot.

Check if customers with OnlineSecurity = Yes have lower churn.

Find the top 3 services (e.g., OnlineBackup, StreamingTV, TechSupport) most associated with churn.

Group customers by Contract and find the average tenure and MonthlyCharges.

Group by PaymentMethod and show the churn percentage for each.

Calculate the total revenue (sum of TotalCharges) from churned vs. active customers.

Encode categorical variables and check the shape of the new DataFrame.

Split the data into training and testing sets for predictive modeling (80%-20%).
